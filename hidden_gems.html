Hidden Gems: Lesser-Known HTML & CSS Features

Discover powerful but underutilized web development features

Lesser-Known HTML Tags

HTML offers many specialized elements that can improve semantics, accessibility, and functionality.

<fieldset> and <legend>

Group related form elements with a fieldset and provide a caption with legend.

<fieldset>
    <legend>Personal Information</legend>
    <label for="name">Name:</label>
    <input type="text" id="name">
    <label for="email">Email:</label>
    <input type="email" id="email">
</fieldset>
Personal Information

<details> and <summary>

Create a disclosure widget that shows additional information when toggled.

<details>
    <summary>Click for more information</summary>
    <p>This is additional content that appears when expanded.</p>
</details>
Click for more information

This is additional content that appears when expanded. It's great for FAQs, instructions, or any content that doesn't need to be immediately visible.

Underutilized CSS Properties

CSS has many powerful properties that are often overlooked but can solve common design challenges.

isolation Property

The isolation property determines whether an element must create a new stacking context.

.isolated-element {
    isolation: isolate;
}

Normal stacking context (z-index: 2)

This element has z-index: 2

This element has z-index: 1 but appears on top due to source order

With isolation: isolate (creates new stacking context)

This element has z-index: 2

This element has z-index: 1 but appears below due to isolation

Filter Effects

.filtered {
    filter: blur(5px) brightness(1.5);
}

Transform: Flip

.flipped-horizontal {
    transform: scaleX(-1);
}
.flipped-vertical {
    transform: scaleY(-1);
}
.flipped-both {
    transform: scale(-1, -1);
}

Dynamic Grid Layouts

CSS Grid offers powerful layout capabilities beyond basic rows and columns.

.dynamic-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
    gap: 1rem;
}

More HTML Gems

  • <output> - Represents the result of a calculation
  • <cite> - Marks a reference to a creative work
  • <code> - Displays a fragment of code
  • <kbd> - Represents keyboard input
  • <samp> - Represents sample output
  • <time> - Represents a specific time or date
  • <mark> - Highlights text for reference

More CSS Gems

  • object-fit - Controls how content fits in containers
  • clip-path - Creates clipping regions
  • shape-outside - Wraps content around custom shapes
  • mix-blend-mode - Blends element with background
  • scroll-snap - Creates scroll snapping effects
  • conic-gradient() - Creates circular gradients
  • gap - Controls spacing in grid and flexbox

Browser Support

Most modern browsers support these features:

  • Fieldset/Legend: All browsers
  • Details/Summary: All modern browsers
  • Meter/Progress: All modern browsers
  • CSS Isolation: All modern browsers
  • CSS Filters: All modern browsers
  • CSS Grid: All modern browsers

Accessibility Benefits

  • Semantic HTML improves screen reader experience
  • Fieldset/legend groups related form elements
  • Abbr provides explanations for abbreviations
  • Details/summary creates accessible disclosures
  • Meter/progress provide accessible status indicators

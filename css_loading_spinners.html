CSS Loading Spinners Collection

Beautiful, customizable loading spinners for your web projects

Why Use Loading Spinners?

Loading spinners provide visual feedback to users during operations that take time to complete. They improve user experience by indicating that content is being processed or loaded.

Best practices for loading indicators:

  • Use them for operations that take more than 0.5 seconds
  • Keep them simple and unobtrusive
  • Match your website's design aesthetic
  • Consider using skeleton screens for content loading
  • Provide text labels when appropriate

CSS Loading Spinners

Here's a collection of 10 unique CSS loading spinners you can use in your projects:

Simple Spinner

A classic circular spinner with a smooth rotation.

<div class="spinner"></div>
.spinner {
  width: 50px;
  height: 50px;
  border: 5px solid #f3f3f3;
  border-top: 5px solid #4a86e8;
  border-radius: 50%;
  animation: spin 1s linear infinite;
}

@keyframes spin {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}

Multi-Layer Spinner

A spinner with multiple rotating layers for a more complex effect.

<div class="spinner">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
.spinner {
  width: 40px;
  height: 40px;
  position: relative;
}

.spinner div {
  position: absolute;
  width: 100%;
  height: 100%;
  border: 4px solid transparent;
  border-top-color: #4a86e8;
  border-radius: 50%;
  animation: spin 1.2s cubic-bezier(0.5, 0, 0.5, 1) infinite;
}

.spinner div:nth-child(1) {
  animation-delay: -0.45s;
}

.spinner div:nth-child(2) {
  animation-delay: -0.3s;
}

.spinner div:nth-child(3) {
  animation-delay: -0.15s;
}

@keyframes spin {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}

Ease Spinner

A smooth spinner with ease-in-out animation.

<div class="spinner"></div>
.spinner {
  width: 50px;
  height: 50px;
  display: inline-block;
  border: 5px solid rgba(255, 255, 255, 0.3);
  border-radius: 50%;
  border-top-color: #4a86e8;
  animation: spin 1s ease-in-out infinite;
}

@keyframes spin {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}

Pulse Spinner

A pulsing effect that expands and contracts.

<div class="spinner"></div>
.spinner {
  width: 50px;
  height: 50px;
  position: relative;
}

.spinner::before,
.spinner::after {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  border-radius: 50%;
  background-color: #4a86e8;
  opacity: 0.6;
  animation: pulse 2s ease-in-out infinite;
}

.spinner::after {
  animation-delay: -1s;
}

@keyframes pulse {
  0%, 100% { transform: scale(0); }
  50% { transform: scale(1); }
}

Double Border Spinner

A spinner with both top and bottom borders animated.

<div class="spinner">
  <div class="spinner-inner"></div>
</div>
.spinner {
  width: 50px;
  height: 50px;
  border-radius: 50%;
  perspective: 800px;
}

.spinner-inner {
  width: 100%;
  height: 100%;
  border-radius: 50%;
  border: 5px solid transparent;
  border-top-color: #4a86e8;
  border-bottom-color: #4a86e8;
  animation: spin 1.5s linear infinite;
}

@keyframes spin {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}

Bouncing Dots

Three dots that bounce in sequence.

<div class="spinner">
  <div></div>
  <div></div>
  <div></div>
</div>
.spinner {
  width: 50px;
  height: 50px;
  display: flex;
  justify-content: space-around;
}

.spinner div {
  width: 12px;
  height: 12px;
  background-color: #4a86e8;
  border-radius: 50%;
  animation: bounce 1.4s ease-in-out infinite both;
}

.spinner div:nth-child(1) {
  animation-delay: -0.32s;
}

.spinner div:nth-child(2) {
  animation-delay: -0.16s;
}

@keyframes bounce {
  0%, 80%, 100% { transform: scale(0); }
  40% { transform: scale(1); }
}

How to Use These Spinners

To use any of these spinners in your project:

  1. Copy the HTML structure for your chosen spinner
  2. Copy the corresponding CSS styles
  3. Add the animation keyframes if needed
  4. Customize colors and sizes to match your design

Pro Tip: You can customize the appearance of these spinners by changing the color values, sizes, animation durations, and border widths to match your website's design system.

Spinner Best Practices

  • Use appropriate size for context
  • Match your brand colors
  • Consider accessibility needs
  • Test performance on mobile devices
  • Provide text alternatives when needed

When to Use Spinners

  • Form submissions
  • Content loading
  • Image processing
  • API requests
  • File uploads

Performance Considerations

  • CSS animations are GPU accelerated
  • Keep animations simple for best performance
  • Test on lower-powered devices
  • Consider reducing motion for accessibility

Accessibility Tips

  • Use ARIA attributes for screen readers
  • Provide text alternatives with sr-only class
  • Respect prefers-reduced-motion media query
  • Ensure sufficient color contrast

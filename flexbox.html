CSS Flexbox: The Complete Guide

Master modern layout techniques with CSS Flexbox

What is Flexbox?

Flexbox (Flexible Box Layout) is a CSS layout module that makes it easier to design flexible and responsive layout structures without using float or positioning. It provides a more efficient way to lay out, align, and distribute space among items in a container.

Flexbox is especially useful for:

  • Creating responsive layouts that work on different screen sizes
  • Distributing space dynamically between elements
  • Aligning content both vertically and horizontally
  • Creating navigation bars, grids, and card layouts

Flexbox Container Properties

To create a flex container, set the display property to flex or inline-flex. This enables a flex context for all its direct children.

display

Defines a flex container:

.container {
  display: flex; /* or inline-flex */
}

flex-direction

Establishes the main axis and direction of flex items:

.container {
  flex-direction: row | row-reverse | column | column-reverse;
}

justify-content

Defines alignment along the main axis:

.container {
  justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;
}

align-items

Defines how flex items are laid out along the cross axis:

.container {
  align-items: stretch | flex-start | flex-end | center | baseline;
}

Flexbox Item Properties

Flex items (the direct children of a flex container) can also have properties that control their flexibility and alignment.

flex-grow

Defines the ability for a flex item to grow if necessary:

.item {
  flex-grow: 4; /* default 0 */
}

flex-shrink

Defines the ability for a flex item to shrink if necessary:

.item {
  flex-shrink: 3; /* default 1 */
}

flex-basis

Defines the default size of an element before the remaining space is distributed:

.item {
  flex-basis: 100px; /* default auto */
}

Practical Flexbox Examples

Let's look at some real-world examples of how Flexbox can be used to create common layouts.

Navigation Bar

Creating a responsive navigation bar is simple with Flexbox:

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.nav-links {
  display: flex;
  gap: 20px;
}

@media (max-width: 768px) {
  .navbar {
    flex-direction: column;
  }
}

Key Benefits of Flexbox

  • Easier vertical and horizontal alignment
  • Ability to reorder elements without changing HTML
  • Responsive design without media queries for simple cases
  • Equal height columns by default
  • Dynamic space distribution between elements

Browser Support

Flexbox is supported in all modern browsers:

  • Chrome: 29+
  • Firefox: 28+
  • Safari: 9+
  • Edge: 12+
  • Opera: 17+

For older browsers, consider using fallbacks or alternative approaches.

Common Use Cases

  • Navigation bars and menus
  • Card layouts and grids
  • Vertical and horizontal centering
  • Sticky footers
  • Form layouts
  • Media objects (image + text)

Best Practices

  • Use flex-wrap for responsive layouts
  • Combine with CSS Grid for complex layouts
  • Use gap property for spacing (instead of margins)
  • Test on various screen sizes
  • Use flex shorthand: flex: grow shrink basis

DOM Manipulation: Complete Guide

Learn how to interact with and manipulate the Document Object Model with JavaScript

What is the DOM?

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the page structure so that programs can change the document structure, style, and content.

Key Concepts:

  • The DOM represents a document as a tree of nodes
  • JavaScript can manipulate all nodes in the tree
  • Changes to the DOM are immediately reflected in the browser
  • The DOM is language-neutral and can be used with any programming language

DOM Node Types

Common DOM node types:

  • Element nodes (tags like <div>, <p>)
  • Text nodes (text content)
  • Comment nodes (HTML comments)
  • Document node (the entire document)
  • Attribute nodes (HTML attributes)

Why Manipulate the DOM?

DOM manipulation allows you to:

  • Change content dynamically
  • Update styles on the fly
  • Respond to user interactions
  • Create single-page applications
  • Build interactive web experiences

Selecting DOM Elements

To manipulate elements in the DOM, you first need to select them. JavaScript provides several methods for selecting elements.

Selection Methods

// Select by ID
const element = document.getElementById('myId');

// Select by class name (returns HTMLCollection)
const elements = document.getElementsByClassName('myClass');

// Select by tag name (returns HTMLCollection)
const paragraphs = document.getElementsByTagName('p');

// Select by CSS selector (returns first match)
const element = document.querySelector('.myClass');

// Select by CSS selector (returns NodeList)
const elements = document.querySelectorAll('.myClass');

Note: querySelector and querySelectorAll are the most versatile methods as they accept any CSS selector.

// JavaScript code for selection
function selectById() {
  const element = document.getElementById('demo-box1');
  element.classList.add('highlight');
}

function selectByClass() {
  const elements = document.getElementsByClassName('highlightable');
  for (let el of elements) {
    el.classList.add('highlight');
  }
}

function selectByQuery() {
  const elements = document.querySelectorAll('.highlightable');
  elements.forEach(el => el.classList.add('highlight'));
}

function resetSelection() {
  const elements = document.querySelectorAll('.dom-element');
  elements.forEach(el => el.classList.remove('highlight'));
}

Modifying Elements

Once you've selected DOM elements, you can modify their content, attributes, and styles.

Content Modification

// Change text content
element.textContent = 'New text content';

// Change HTML content
element.innerHTML = '<strong>New HTML content</strong>';

// Get content
const content = element.textContent;
const htmlContent = element.innerHTML;

Attribute Modification

// Set attributes
element.setAttribute('id', 'newId');
element.setAttribute('class', 'newClass');

// Get attributes
const id = element.getAttribute('id');
const className = element.getAttribute('class');

// Remove attributes
element.removeAttribute('id');

// Check for attributes
const hasClass = element.hasAttribute('class');

Creating and Removing Elements

You can dynamically create new elements and add them to the DOM, or remove existing elements.

Creating Elements

// Create new element
const newElement = document.createElement('div');

// Add content and attributes
newElement.textContent = 'New div element';
newElement.setAttribute('class', 'my-new-div');

// Append to existing element
parentElement.appendChild(newElement);

// Insert before existing element
parentElement.insertBefore(newElement, existingElement);

// Insert adjacent HTML
element.insertAdjacentHTML('beforebegin', '<div>New element</div>');

Removing Elements

// Remove element
element.remove();

// Remove child element
parentElement.removeChild(childElement);

// Replace element
parentElement.replaceChild(newElement, oldElement);

Event Handling

Events are actions that happen in the browser, such as clicks, mouse movements, or keyboard presses. You can listen for these events and execute code in response.

Event Listeners

// Add event listener
element.addEventListener('click', function(event) {
  // Code to execute when element is clicked
  console.log('Element clicked!');
});

// Remove event listener
function handleClick() {
  console.log('Clicked!');
}
element.addEventListener('click', handleClick);
element.removeEventListener('click', handleClick);

// Common event types
// click, dblclick, mouseover, mouseout, mousedown, mouseup
// keydown, keyup, keypress
// focus, blur, change, submit
// load, resize, scroll

Event Object

When an event occurs, the browser creates an event object containing information about the event:

element.addEventListener('click', function(event) {
  console.log(event.target); // Element that triggered the event
  console.log(event.type);   // Event type (e.g., 'click')
  console.log(event.clientX, event.clientY); // Mouse coordinates
});

DOM Traversal

DOM traversal involves moving between elements in the DOM tree, accessing parent, child, and sibling elements.

Traversal Methods

// Parent nodes
const parent = element.parentNode;
const parentElement = element.parentElement;

// Child nodes
const children = element.childNodes; // Includes text nodes
const children = element.children;   // Only element nodes
const firstChild = element.firstChild;
const lastChild = element.lastChild;

// Sibling nodes
const nextSibling = element.nextSibling;
const previousSibling = element.previousSibling;
const nextElementSibling = element.nextElementSibling;
const previousElementSibling = element.previousElementSibling;

Practical Example

// Find all paragraphs and add a class to their parent
const paragraphs = document.querySelectorAll('p');
paragraphs.forEach(p => {
  p.parentElement.classList.add('has-paragraph');
});

// Find next sibling and change its color
const element = document.getElementById('myElement');
const next = element.nextElementSibling;
if (next) {
  next.style.color = 'red';
}
